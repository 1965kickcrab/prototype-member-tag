## 정책 의사결정 완료 항목

1. 태그명 글자수 길이 제한
   - 확인된 사실: 현재 구현 상으로 태그 입력, 새 태그 추가, 태그명 수정 시 앞뒤 공백 제거 및 연속 공백을 한 칸으로 정리는 수행하지만 최대 글자수 제한은 두고 있지 않습니다.
   - 결정된 정책: 회원 태그명을 **최대 10자**까지 허용하도록 결정되었습니다. (10자 초과 입력 시 입력 제한)
   - 상태: 결정 완료

2. 다중 태그 필터 조건 (AND vs OR)
   - 확인된 사실: 회원 목록, 유치원 예약 목록, 호텔링 예약 목록 필터에서 다중 태그를 선택할 경우, 선택된 모든 태그가 대상 회원(보호자 태그 + 반려견 태그)에 전부 포함된 경우에만 결과에 노출하도록 구현되어 있습니다 (AND 조건).
   - 결정된 정책: 기존의 **AND 조건**을 그대로 유지하기로 결정되었습니다.
   - 상태: 결정 완료

3. 대체 태그 미지정 시 반려견 정보 내 태그 일괄 제거
   - 확인된 사실: 사용 중인 태그를 삭제할 때 대체 태그를 "선택 안함"으로 지정하고 저장할 경우, 해당 태그를 연결해 사용 중이던 회원들의 반려견 정보에서 해당 태그 항목이 일괄 유실(제거)됩니다.
   - 결정된 정책: 기존의 **일괄 제거 전파 방식**을 유지하기로 결정되었습니다.
   - 상태: 결정 완료
